ＦＴオプション【ファイルローダー】README


【名　　称】　ＦＴ用オプション:ファイルローダー  Version 0.3
【登 録 名】　ZFLR03.LZH
【制作者名】　山本三七男（たろサ） QGB03241
【動作環境】　ザウルスＰＩ−６０００〜４５００



　このプログラムはＦＴのオプションプログラムです。ＦＴが無いと実行がで
きません。ＦＴにインストールすると、次のことが行えるようになります。

　・生のザウルス（何もアドインが入っていないザウルス）へのファイル転送

　　　ができます。

　必要な方だけ、ＦＴのメニューに組み込んでお使い下さい。

【今回のバージョンアップ内容】
　・ザウルスのリンクを自動的に切断するようにした。
　・終了時にテンポラリファイルを消去するようにした。
