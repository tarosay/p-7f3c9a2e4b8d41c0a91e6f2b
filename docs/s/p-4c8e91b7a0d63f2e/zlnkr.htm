ＦＴオプション【ザウルスでリンク】README

【名　　称】　ＦＴ用オプション:ザウルスでリンク  Version 0.02
【登 録 名】　ZLNK002.LZH
【制作者名】　山本三七男（たろサ） QGB03241
【動作環境】　ザウルスＰＩ−６０００〜４５００



　このプログラムはＦＴのオプションプログラムです。ＦＴが無いと実行がで
きません。ＦＴにインストールすると、次のことが行えるようになります。

　　・ザウルスのパソコンリンク機能を利用して、ザウルス同士のファイルの
　　　交換や削除ができます。

　　・ＰＩ−４０００等のアドインが使えない機種に、アドインを転送するこ
　　　ともできます。（最も、実行できませんが(^^;）
